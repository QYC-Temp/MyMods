# 金币互赠 Gold Gift

《杀戮尖塔 2》联机 mod：在多人游戏中向队友赠送金币。

## 功能

- 联机局中向任意队友赠送任意金额的金币
- 双方金币实时同步，不会出现不同步
- 金币不足时会就地提示，不会误扣

## 使用要求

- 《杀戮尖塔 2》
- **房间内所有玩家都必须安装此 mod**

## 安装

1. 解压下载的压缩包
2. 把 `GoldGift` 文件夹整个放到游戏目录的 `mods` 文件夹下
3. 启动游戏即可

## 使用方法

1. 进入联机房间
2. 点击队友头像打开详情面板
3. 点左下角的 **Gift Gold** 按钮
4. 输入金额，确认即可

## 已知限制

- 单人模式下按钮不会显示
- 所有玩家必须使用相同版本的 mod
